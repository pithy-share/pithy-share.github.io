    抗刷老指南-安德鲁.韦尔

    衰老理论¶

    1. 细胞分裂次数有限 端粒损失
    2. 炎症
    3. 焦糖化

    4. 氧化应激

    5. 自由基

    6. 吸烟是一种最集中的自由基传递方式

    措施总原则¶

    1. 好的营养， 足够的脂肪酸
    2. 足量水
    3. 避免肥皂等刺激皮肤的
    4. 避免阳光照射
    5. 抗炎

    身体（一）：预防疾病¶

    检查¶

    • 高血压
    • 胆固醇
    • 血糖

    两点忠告¶

    • 不吸烟
    • 体重

    身体（二）：抗炎饮食¶

    总则¶

    • 力争多样化
    • 多吃新鲜食物
    • 少吃加工过的食物和快餐
    • 吃大量水果和蔬菜

    炎症基本症状¶

    • 红
    • 热
    • 肿
    • 痛

    脂肪¶

    增加Omega-3 脂肪酸的摄入¶

    Omega-3

    • 绿叶蔬菜

    • 几种坚果

    • 种子（核桃，亚麻，大麻）

    • 几种植物油（豆油，芥花油）

    • 海里的植物

    • 寒冷水域中的含油鱼类（鲑鱼，沙丁鱼，鲱鱼，鲭鱼，黑鳕鱼，青鱼）鱼油补充剂

    减少Omega-6脂肪酸的摄入¶

    Omega-6

    • 炸薯条
    • 薄脆饼干
    • 糖果

    排除促进发炎的脂肪¶

    • 部分氢化油
    • 植物起酥油
    • 人造黄油
    • 油炸食品
    • 变味油脂
    • 多不饱和植物油
    • 不把油加热到冒烟
    • 应当使用压榨或冷压的油

    碳水化合物¶

    • 少吃高GL的食物
      • 少吃面包，土豆，薄脆饼干，油炸食品，各种甜点和饮料
      • 多吃全谷物，豆类，甘薯，笋瓜和其他蔬菜
      • 适量吃一些温带水果（浆果，樱桃，苹果，梨）
      • 少吃热带水果（菠萝，芒果，木瓜）
    • 少吃精加工的食品
    • 少吃快餐，最好不吃
    • 少吃用各种面粉制作的食物
    • 不吃用高左旋糖玉米糖浆制作的食物

    蛋白质¶

    • 少吃兽肉，禽肉和其他动物食品
    • 多吃植物蛋白质：大豆及其制品，其他豆类，全谷物，种子，坚果
    • 少受污染的鱼

    微量营养素 维生素和矿物质¶

    多吃蔬菜和水果

    花青素¶

    • 蓝莓，黑莓，树莓
    • 樱桃，红葡萄，石榴，红甘蓝，甜菜
    • 深绿色蔬菜

    植物多酚¶

    • 白茶，绿茶，乌龙茶

    类胡萝卜素¶

    • 红，橙，黄色的水果和蔬菜（芒果，桃，胡萝卜，甘薯，菠菜，南瓜等）

    其他¶

    • 甘蓝科蔬菜 I-3-C
    • 西兰花 萝卜硫素
    • 生姜

    身体（三）补充剂¶

    多种维生素，不能空腹¶

    • 不应包含预先成形的维生素A
    • 多种类胡萝卜素
    • 维生素E 复合天然生育酚 生育三烯醇
    • 绝大部分B族维生素 叶酸至少400微克，维生素B12至少50微克，其他各种B族维生素为每种50毫克
    • 维生素C不必太多
    • 至少400国际单位的维生素D
    • 不应含铁
    • 不超过200微克的硒
    • 钙 最好是柠檬酸钙

    抗氧化剂¶

    • Co-Q-10 辅酶Q
    • 葡萄籽提取物和松树皮提取物
    • α-硫辛酸

    抗炎补充剂¶

    • 生姜和姜黄
    • 阿司匹林

    充实体能¶

    • 脱氢表雄酮 DHEA

    增强免疫系统¶

    • 黄芪
    • 菌类 香菇等

    解毒¶

    • 奶蓟

    补充能量¶

    • 人参
    • 刺五加
    • 红景天
    • 冬虫夏草

    提高性能力¶

    • 睾酮
    • 人参（亚洲参）
    • 醉茄
    • 红景天

    身体（四）锻炼身体¶

    有氧运动¶

    • 步行
    • 游泳
    • 骑自行车

    力量锻炼¶

    灵活性与平衡性¶

    • 瑜伽
    • 太极拳

    身体（五）休息与睡眠¶

    • 休息很重要
    • 午后小睡
    • 尽可能晒太阳 注意辐射
    • 晚上睡觉前，1小时暗淡光线
    • 天然安眠药 缬草 褪黑素

    身体（六）亲密接触与性¶

    心理（一） 压力¶

    身体对压力作出反应的肾上腺激素-皮质醇会直接损害大脑内掌管记忆和情绪的那些神经细胞。

    交感神经系统的过分活跃事慢性病的一大原因

    放松反应¶

    • 呼吸训练（慢，深，静）
    • 冥想（集中在呼吸和体内感受上）
    • 视觉想象（想象去过而感到及其快乐，安静和宁静的地方）
    • 身体按摩

    心理（二）思想，情绪，态度¶

    抑郁和焦虑¶

    抑郁的根源：习惯性产生自我贬抑的想法和孤僻感

    焦虑症的根源：自认为无能为力或无法应对日常生活带来的调整

    改变思维模式¶

    • 创新的精神疗法 塞利格曼的学习乐观
    • 灵活和幽默（学会不计较得失）
    • 佛教心理学 正念减压法

    心理（三）记忆¶

    • 保持社会联系和智力联系
    • 教育
    • 银杏提取物
    • 乙酰左旋肉碱
    • 磷脂酰丝氨酸（PS）
    • 学习新操作系统和外语

    精神（一）不变的实质¶

    • 专注于呼吸
    • 于自然相连
    • 与哪些人在一起更愉悦
    • 把花带回家
    • 聆听音乐
    • 争取跟与你疏远了的人重新建立联系，体现宽容之心
    • 做一些服务性工作，助人

    精神（二）道德遗嘱¶

    自己有哪些价值观和生活经验要留给别人
