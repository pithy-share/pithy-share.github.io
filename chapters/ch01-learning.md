# 第 1 章　机器到底怎么"学"

> **第一性原理 #1：学习 = 参数更新。** 所有机器学习、所有大模型训练，剥到底都是同一件事：调参数，让误差变小。

## 损失函数：把"错得多离谱"打成一分

参数从 2 个涨到几千亿个、误差公式从简到繁，"调参数降误差"这个动作从没变过。这一章用最小例子走通它。

有批数据点大致排成直线 `y = w·x + b`，`w`、`b` 未知——这就是要学的参数。先瞎猜（比如 `w=b=0`），再量化它有多烂：**损失函数**。最常用的是**均方误差（MSE）**：

```
L = (1/n) · Σ (w·xᵢ + b − yᵢ)²
```

平方干两件事：让正负误差不抵消（错 +3 和 −3 都算"错 3"），并重罚大错。分数越高越烂，学习就是让它往下掉。

## 梯度下降：往下坡走一步

`L` 是 `w`、`b` 的函数。**梯度**是"脚下的坡度"——指明 `L` 沿哪个方向涨得最快。要下坡，就往反方向走：

```
w ← w − 学习率 · ∂L/∂w
b ← b − 学习率 · ∂L/∂b
```

1D 直线 + MSE 的梯度直接可算：

```
∂L/∂w = (2/n) · Σ xᵢ · (w·xᵢ + b − yᵢ)
∂L/∂b = (2/n) · Σ       (w·xᵢ + b − yᵢ)
```

记不住公式没关系，记住意思：**梯度 = "每个参数该往哪调、调多少"的指示牌**。学习率是每步多大——太大跨过谷底，太小磨蹭，就一个要调的数。

## 反向传播：链式法则分摊"责任"

`y = w·x + b` 只有一步，梯度一眼能算。真实模型是一长串运算，参数要经过许多步骤才影响最终输出。这时每个参数的梯度靠**链式法则**：A 影响 B、B 影响 C，则 A 对 C 的影响 = (A→B) × (B→C)，一层乘一层。

最小链条，损失经三步才碰到 `w`：

```
z = w·x + b   →   â = σ(z)   →   L = (â − y)²
```

```
∂L/∂w = ∂L/∂â · ∂â/∂z · ∂z/∂w
```

每步局部梯度都好算，乘起来就是 `w` 对最终损失的影响。**反向传播（backpropagation）** 就是从损失出发、从后往前、一层层乘局部梯度，把误差分摊到每个参数。链条变成几十上百层，就是下一章的**神经网络**——机制还是这条链式法则，没换过。

## 动手：看参数自己滑到对的位置

造一批满足 `y = 2x + 3` 的带噪声数据，从 `w=b=0` 开始，让机器自己学出这条直线。

```python
import random
random.seed(0)            # 固定随机种子，你跑出来的数和我一模一样

true_w, true_b = 2.0, 3.0
xs = [0.1 * i for i in range(-20, 21)]                  # x: -2.0 ~ 2.0
ys = [true_w * x + true_b + random.gauss(0, 0.2) for x in xs]

w, b = 0.0, 0.0
lr = 0.01
for step in range(1000):
    grad_w = sum(2 * (w*x + b - y) * x for x, y in zip(xs, ys)) / len(xs)
    grad_b = sum(2 * (w*x + b - y)       for x, y in zip(xs, ys)) / len(xs)
    w -= lr * grad_w
    b -= lr * grad_b
    if step % 200 == 0:
        loss = sum((w*x + b - y)**2 for x, y in zip(xs, ys)) / len(xs)
        print(f"step {step:4d}  w={w:.3f}  b={b:.3f}  loss={loss:.4f}")
print(f"学到的直线：y = {w:.3f}·x + {b:.3f}")
```

```
step    0  w=0.056  b=0.060  loss=13.9584
step  200  w=2.006  b=2.933  loss=0.0513
step  400  w=2.012  b=2.983  loss=0.0485
step  600  w=2.012  b=2.984  loss=0.0485
step  800  w=2.012  b=2.984  loss=0.0485
学到的直线：y = 2.012·x + 2.984
```

`w`、`b` 从 0 自己滑向 `2`、`3`——我们从没给过答案，**这就是学习在发生**。把 `lr` 调到 `0.5`，看它一步跨过谷底、来回震荡。

## 底座

> 学习 = 损失函数量化误差 + 梯度下降调参数 + 反向传播算梯度。
